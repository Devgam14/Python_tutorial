Version : pip --version
install packages : pip install <package_name>
uninstall packages : pip uninstall <package_name>
multiple packages : pip install <name1>  <name2> <name3>
getting there versions just like package.json : pip freeze > requirements.txt 
using requirement.txt to uninstall packages : pip uninstall -r requirements.txt
using requirement.txt to install packages : pip install -r requirements.txt
